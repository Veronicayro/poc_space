# Architecture Agent Assets

This folder contains reusable architecture assets consumed by Copilot agents and skills.

```text
architecture/context/   # governance rules, architectural principles, constraints
architecture/schemas/   # artifact contracts and output structures
architecture/prompts/   # reusable execution prompts
architecture/migration/ # path mapping and migration notes
```

Do not place Copilot-discoverable agents or skills here. Use `.github/agents/` and `.github/skills/`.
